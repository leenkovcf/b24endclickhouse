# b24endclickhouse

Локальное веб-приложение: выгружает данные из Bitrix24 через BI Connector и загружает в ClickHouse.

## Возможности

- **30+ сущностей**: сделки, лиды, контакты, компании, задачи, звонки, КЭДО, склад, смарт-процессы и др.
- **Фильтры**: воронки, стадии (с группировкой по воронке), выбор конкретных полей
- **Русские названия полей** — подтягиваются через REST API
- **Прогресс в реальном времени** — строки по каждому дню
- **Расписание**: ежедневно / еженедельно (пн) / ежемесячно (1-го), время МСК
- **Смарт-процессы**: автозагрузка через REST Webhook или ручное добавление
- Настройки и состояние фильтров сохраняются между сессиями

## Требования

- Python 3.10+
- Bitrix24 с включённым BI Connector
- ClickHouse (Yandex Cloud или self-hosted)

## Установка и запуск

**Windows:**
```
install.bat   # установка зависимостей
start.bat     # запуск → http://127.0.0.1:8000
```

**Вручную:**
```bash
python -m venv venv
venv\Scripts\activate        # Windows
# source venv/bin/activate   # Linux/macOS
pip install -r requirements.txt
uvicorn main:app --host 127.0.0.1 --port 8000
```

## Настройка

**Bitrix24 BI-ключ:** CRM → BI-аналитика → Управление ключами

**REST Webhook** *(необязательно)*: нужен для фильтров по воронкам/стадиям и автозагрузки смарт-процессов.
Создаётся в: Профиль → Разработчикам → Входящий вебхук (права: CRM).

**ClickHouse:** поддерживается Yandex Cloud (порт 8443, SSL) и self-hosted (8123/9000).

## Использование

1. **Подключение** → заполнить данные → **Проверить подключение**
2. **Отправка данных** → сущность → период → фильтры → **Запустить выгрузку**
3. **Расписание** → периодичность + время МСК + глубина в днях → **Сохранить**

Таблицы в ClickHouse создаются автоматически. Типы колонок определяются автоматически (`Int64`, `Float64`, `Date`, `DateTime`, `String`). Данные **дописываются** (APPEND) — при необходимости дедупликации используйте `DISTINCT` или `ReplacingMergeTree` на стороне ClickHouse.
