Подключение
Каждый источник настраивается независимо. ClickHouse обязателен; Bitrix24 и Ozon — опциональны.
ClickHouse обязательно ▾

Куда писать данные. Если у вас Yandex Cloud — хост заканчивается на .mdb.yandexcloud.net, порт 8443. Для локального ClickHouse без SSL — порт 8123.

Bitrix24 опционально ▾

Для выгрузки CRM-данных через BI-коннектор. Если работаете только с Ozon — оставьте пустым, эта секция не будет блокировать приложение.

Смарт-процессы и кастомные сущности

Смарт-процессы уникальны для каждого портала. Добавьте вручную: код таблицы вида crm_dynamic_items_105 и название.

Ozon опционально, несколько кабинетов ▾

Можно подключить несколько личных кабинетов Ozon (например, два магазина). Каждый — отдельным аккаунтом. Ключи получаются в Seller API: Настройки → Seller API → API ключи. Для рекламной аналитики — отдельные ключи Performance API в разделе Реклама → Performance API.

Аккаунт Ozon ×
Seller API обязательно

Настройки → Seller API → API ключи → создать ключ с нужными правами

Performance API опционально, для рекламы

Реклама → Performance API → API ключи. Нужно только если хотите выгружать статистику рекламных кампаний

Автоматическая выгрузка по расписанию — работает пока приложение запущено
Проверка подключения...
Как работает: В указанное время приложение выгружает данные за последние N дней назад в ClickHouse — по очереди для каждой выбранной сохранённой конфигурации. Глубина выгрузки задаётся ниже — по умолчанию 1 день для ежедневной, 7 для еженедельной, 30 для ежемесячной.
Ежедневно — каждый день в указанное время. Еженедельно — каждый понедельник. Ежемесячно — 1-го числа каждого месяца.
Сначала создайте конфигурации на вкладке «Отправка данных» (с нужными воронками, стадиями и полями) — затем выберите их здесь.
⚠ Расписание активно только пока приложение запущено.

О приложении
Описание, инструкция и примечания

Приложение выгружает данные из Bitrix24 через BI-коннектор и загружает их в ClickHouse для дальнейшего анализа в Yandex DataLens или других BI-инструментах.

Как начать работу

  • Откройте Подключение и заполните данные Bitrix24 и ClickHouse.
  • Нажмите Проверить подключение — оба должны быть зелёными.
  • Перейдите в Отправка данных, выберите сущность, период и запустите выгрузку.
  • Для автоматических выгрузок настройте Расписание.

Как получить BI-ключ Bitrix24

  • CRM → BI-аналитика → Управление ключами
  • Создайте новый ключ или скопируйте существующий

Примечания

  • Скорость выгрузки зависит от скорости интернета и Bitrix24 API, а не от мощности ПК.
  • Таблицы в ClickHouse создаются автоматически при первой выгрузке.
  • Настройки хранятся в файле config.json в папке приложения.
  • Расписание работает пока приложение запущено.
